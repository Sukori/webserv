✅ Test GET – Documents Statiques

Accédez aux pages HTML et fichiers statiques ci-dessus.

curl http://localhost:8081/
curl http://localhost:8081/about.html

📤 Test POST – Téléchargement Fichier

Utilisez le formulaire ci-dessous ou curl pour tester l'upload:

curl -X POST --data-binary @file.txt http://localhost:8081/uploads/
curl -F "file=@document.pdf" http://localhost:8081/uploads/

→ Aller au formulaire

🗑️ Test DELETE – Suppression Fichier

Supprimer le fichier /download-me.txt:

curl -X DELETE http://localhost:8081/download-me.txt

Attendu: 204 No Content ou 200 OK si succès.

📁 Test Directory Listing / Autoindex

Listage personnalisé du dossier /files/:

curl http://localhost:8081/files/

Attendu: HTML ou liste texte des fichiers du répertoire.

↩️ Test Redirections HTTP (return)

La location /tests est configurée pour rediriger:

curl -L http://localhost:8081/tests
curl -v http://localhost:8081/tests  # Voir les headers 301/302

⚙️ Test CGI / Scripts Dynamiques

Sujets aux binaires python3 et php-cgi:

  • index.php (depuis /www/php/)
  • curl http://localhost:8081/index.py (Python)

❌ Test Pages d'Erreur

Testez les erreurs HTTP 4xx/5xx:

curl http://localhost:8081/nonexistent.html    # 404
curl -X PUT http://localhost:8081/file.txt      # 405
curl -X POST http://localhost:8081/about.html   # 405 (GET only)

🔌 Test Multi-Port

Trois serveurs écoutent:

  • Port 8080: root /core (minimal)
  • Port 8081: root / + locations complexes (main)
  • Port 8082: root /core (minimal)
curl http://localhost:8080/
curl http://localhost:8081/
curl http://localhost:8082/
