Formulaire POST – Upload Fichier

Choisir un fichier à télécharger

Alternative: Ligne de Commande

Utilisez curl pour tester manuellement:

# Tester avec un fichier binaire
echo "Contenu test" | curl -X POST --data-binary @- http://localhost:8081/uploads/test.txt

# Tester avec un formulaire multipart/form-data
curl -F "file=@/path/to/file.txt" http://localhost:8081/uploads/

# Vérifier que le fichier a été créé
curl http://localhost:8081/uploads/

Spécifications POST

  • Endpoint: http://localhost:8081/uploads/
  • Méthode: POST
  • Content-Type: application/x-www-form-urlencoded ou multipart/form-data
  • Réponse attendue: 201 Created ou 200 OK
  • Lieu sauvegarde: /uploads/ du serveur

Tests Automatisés

# Test 1: Fichier simple texte
echo "Hello Webserv" > /tmp/test.txt
curl -X POST --data-binary @/tmp/test.txt http://localhost:8081/uploads/
curl http://localhost:8081/uploads/  # Vérifier présence

# Test 2: Fichier volumineux (test client_max_body_size)
dd if=/dev/zero bs=1M count=100 of=/tmp/bigfile.bin
curl --upload-file /tmp/bigfile.bin http://localhost:8081/uploads/

# Test 3: Erreur 413 si dépassement client_max_body_size
curl -X POST --data "$(head -c 100000000 /dev/zero | base64)" http://localhost:8081/uploads/
